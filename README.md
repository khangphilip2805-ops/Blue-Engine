**Blue-Engine** is a modern, high-performance game engine designed to bridge the gap between **simplicity for beginners** and **power for professionals**. Built with a focus on clean architecture, cross-platform compatibility, and low-level optimization.

---

## ✨ Key Features

* **User-Centric API:** Designed to be intuitive for those just starting their game dev journey.
* **Professional Toolset:** Advanced memory management and rendering capabilities for experienced developers.
* **Cross-Platform Core:** Engineered for consistent performance across multiple operating systems.
* **Modular Design:** A flexible architecture that allows for easy extension and customization.

## 🛠 Tech Stack

* **Language:** C++20
* **Graphics APIs:** DirectX 12 & Vulkan support
* **Architecture:** Layer-based system with dedicated Render Hardware Interface (RHI)
* **UI:** Integrated ImGui for editor and debugging tools

## 🚀 Getting Started

### Prerequisites
Ensure you have the following installed:
* A C++20 compatible compiler (MSVC, Clang, or GCC)
* CMake 3.20 or higher

### Installation
```bash
# Clone the repository
git clone [https://github.com/your-username/Blue-Engine.git](https://github.com/your-username/Blue-Engine.git)

# Navigate to the project directory
cd Blue-Engine

# Build the project
mkdir build && cd build
cmake ..
cmake --build .
